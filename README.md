# SaaS Metrics Dashboard

An interactive dashboard for analysing key SaaS business metrics built with Python and Streamlit.

## Features
- Upload any SaaS customer CSV file
- Automatically calculates MRR, churn rate and customer count
- Revenue breakdown by plan type
- Customer growth over time chart

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## How to Run Locally
pip install streamlit pandas plotly
streamlit run app.py
